### PIR Motion Sensor (AS312)
<https://m5stack.com/products/pir-module>
```python
from m5stack import *
from m5ui import *
from uiflow import *
import time
import unit

setScreenColor(0x222222)
pir0 = unit.get(unit.PIR, unit.PORTB)

while True:
  if pir0.state:
    setScreenColor(0xff0000)
    wait(1)
  else:
    setScreenColor(0x3333ff)
    wait(1)
```
